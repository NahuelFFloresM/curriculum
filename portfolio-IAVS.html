Khan Academy

Sitio web con herramientas de IA para ahorrar tiempo y facilitar la enseñanza mediante la generación de textos, ejercicios y actividades didácticas. Posee muchas herramientas para ayudar en el aula, tanto para iniciar en un tema, relación con la vida real, textos informativos, tareas, etc. Limitado al conocimiento que solo ofrece khan, no permite el uso de archivos o guías personales, por lo que la clase se adapta a lo que le parece correcto. Si bien depende de internet, todo se puede exportar para trabajar en clase. Por ejemplo, aquí dejo un texto informativo, creado con la herramienta, sobre Teoría de Grafos para usar como recurso en la clase o charla.

NotebookLM

Herramienta de google para resumir textos, crear tarjetas didácticas, entre otras opciones, en base a un “cuaderno” creado por el profesor/estudiante. Este cuaderno se carga con material personal o buscado por internet. Permite tener una buena base teórica para dar un tema o concepto en clase, también permite tener un chat para pedir cosas específicas. El servicio básico sirve para crear una fuerte base teórica para dar en clase. Limitado por la parte paga, no se puede generar ejercicios o actividades para darle a los alumnos para resolver. Solo permite la parte “teórica” del conocimiento. Por ejemplo, dejo un link a un mapa conceptual armado con la herramienta, basada en 10 libros sobre teoría de grafos que encontró en internet y los agregó al cuaderno.
